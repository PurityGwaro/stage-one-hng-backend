API Search Instructions

You can make a GET request with the these two parameters:

  • slack_name: The Slack name of the user you want to search for.
  • track: The track you want to search for.

For example, you can make a request like this(add any to the end of the URL above):

  • /api?slack_name=Purity Gwaro&track=frontend
  • /api?slack_name=purity gwaro&track=backend

This will search for my profile. Anything other than that will return an error
